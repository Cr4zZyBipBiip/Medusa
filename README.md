# Medusa
An rshell manager for the game Grey Hack

![Main](https://i.imgur.com/yNqr3FQ.png)

### Features
- General Management
  - List Processes
  - Kill Process
  - Launch a Shell
  - Delete a Client
  - Delete Logs on Clients
  - Grabbing of User / Bank and Mail Passwords
- AutoSpreading
- Auto Cracking of NPC Passwords
